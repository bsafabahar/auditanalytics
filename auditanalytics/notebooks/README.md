# Audit Analytics - Jupyter Notebooks

This directory contains Jupyter notebooks for all 13 chapters of "Audit Analytics: Data Science for the Accounting Profession".

## Notebooks

1. `ch_1_aud_fs.ipynb` - Fundamentals of Auditing Financial Statements
2. `ch_2_statistics_analytics.ipynb` - Foundations of Audit Analytics
3. `ch_3_acct_transactions.ipynb` - Analysis of Accounting Transactions
4. `ch_4_planning.ipynb` - Risk Assessment and Planning
5. `ch_5_analytical_review_technical.ipynb` - Analytical Review: Technical Analysis
6. `ch_6_analytical_review_intell.ipynb` - Analytical Review: Intelligence Scanning
7. `ch_7_design_of_audit.ipynb` - Design of Audit Programs
8. `ch_8_interim_compliance.ipynb` - Interim Compliance Tests
9. `ch_9_substantive.ipynb` - Substantive Tests
10. `ch_10_SOX.ipynb` - Sarbanes-Oxley Engagements
11. `ch_11_block_fraud.ipynb` - Blockchains, Cybercrime and Forensics
12. `ch_12_special.ipynb` - Special Engagements: Forecasts and Valuation
13. `ch_13_simulation.ipynb` - Simulated Transactions for Auditing Service Organizations

## Running the Notebooks

```bash
# Install the package
pip install -e .

# Start Jupyter
jupyter notebook

# Navigate to auditanalytics/notebooks/ and open any notebook
```

## Requirements

All required packages are listed in `requirements.txt`. Make sure to install them before running the notebooks.
